# Task board

### Summary

Build TaskBoard, a single-page React application where users can register, log in, and manage tasks
(create, view, edit, filter, search, sort, delete).

# Technologies Used

VITE , React, TypeScript, React Router, Tailwind CSS.

# Routing

- `/` - Home page (displays a welcome message and navigation links)
- `/register` - User registration page
- `/login` - User login page
- `/tasks` - Task management page (protected route, accessible only to authenticated users)
- `/tasks/:id` - Task detail page (view and edit a specific task)
